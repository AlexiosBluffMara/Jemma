ENVIRONMENT CONSTRAINT EXPLANATION:
================================================================================

Why these instructions instead of live execution from the agent?

The GitHub Copilot CLI agent runs in a sandboxed environment that:
  ✓ CAN access files
  ✓ CAN read/write local files
  ✓ CAN search code
  ✗ CANNOT execute Windows shell commands (requires pwsh.exe PowerShell 7+)
  ✗ CANNOT run Python subprocesses directly (Pylance MCP times out)
  ✗ CANNOT use bash/sh (Windows environment, no WSL in this context)

Solution: Pre-write scripts that can be executed locally using:
  • Windows PowerShell 5.x (included with Windows)
  • Python 3 (available on this machine)
  • Native Windows commands (cmd.exe, wmic, netsh)

These scripts work because they use tools natively available on the local
Windows machine, not through the agent infrastructure.
