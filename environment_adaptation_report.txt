================================================================================
TROUBLESHOOTING
================================================================================

If execution fails:

COMMON ISSUES:

1. Python not in PATH
   Solution: Use full path: C:\PythonXX\python.exe run_workflow_now.py

2. Pip install fails (network issue)
   Solution: Check internet connection, retry with: python -m pip install -e . --verbose

3. Tests fail
   Solution: This may be expected. Check test output in workflow_report.txt
   The workflow still succeeds if Step 6 exit code is 1 (failures) vs -1 (error)

4. Permission denied
   Solution: Run as Administrator or check directory permissions

5. Disk space full
   Solution: Ensure ~300 MB free disk space before running
