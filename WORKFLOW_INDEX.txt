================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

ISSUE: Nothing happens when I double-click run_workflow_now.bat
FIX: Ensure Python is in your PATH
     Alternative: Open Command Prompt in the directory and run manually

ISSUE: "python: command not found"
FIX: Python not in PATH
     Use full path: C:\PythonXX\python.exe run_workflow_now.py

ISSUE: Installation hangs or times out
FIX: Check internet connection
     Retry manually: python -m pip install -e . -v (verbose mode)

ISSUE: Tests fail (exit code 1)
FIX: This may be expected (tests ran but some failed)
     Review test output in workflow_report.txt to see which tests failed
     Exit code 1 = tests ran successfully (some failed)
     Exit code -1 = error occurred

ISSUE: Permission Denied
FIX: Run Command Prompt as Administrator
     Right-click, select "Run as administrator"

ISSUE: Disk space error
FIX: Free up ~300 MB of disk space
     OR use a different drive with more space

ISSUE: Git status shows many files modified
FIX: This shouldn't happen
     Run: git status --short (review what's changed)
     Run: git clean -fd (remove untracked files)
