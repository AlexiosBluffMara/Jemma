================================================================================
WHAT IF SOMETHING GOES WRONG?
================================================================================

PROBLEM: "python: command not found"
SOLUTION: Python not in PATH
FIX: Use full path to Python (e.g., C:\Python311\python.exe run_workflow_now.py)
OR: Add Python to PATH (run: python -m pip install --upgrade pip)

PROBLEM: Pip install takes more than 5 minutes or times out
SOLUTION: Network issue or large package size
FIX: Check internet connection, try again, or increase timeout in script

PROBLEM: Tests fail (Step 6 exit code 1)
SOLUTION: This may be expected
ACTION: Review test output in workflow_report.txt to see which tests failed
NOTE: Exit code 1 means tests ran (success) but some failed
      Exit code -1 means an error occurred (failure)

PROBLEM: Permission Denied
SOLUTION: Insufficient write permissions
FIX: Run as Administrator or check directory permissions

PROBLEM: Disk space error
SOLUTION: Not enough space for ~300 MB of packages
FIX: Free up space or use a different drive

PROBLEM: Git status shows other files modified
SOLUTION: Script didn't properly clean up
FIX: Run "git clean -fd" to remove untracked files
      Review workflow_report.txt to see what was done
